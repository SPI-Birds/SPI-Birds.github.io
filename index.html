Welcome to SPI-Birds Network & Database, a resource hub for code and data related to studies of populations of individual birds.

Please visit the main project website spibirds.org.

All SPI-Birds resources can be found in GitHub repositories, including
protocols and pipelines for generating a standard data format for bird data.
